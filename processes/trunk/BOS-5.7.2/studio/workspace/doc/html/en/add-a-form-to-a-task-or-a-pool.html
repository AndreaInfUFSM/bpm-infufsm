Add a form to a task or a pool

(Category: Forms Modeling)

A form of one or more pages can be added to a pool or to any human task in a process. You can add three type of forms at the pool level :

  • Entry forms (called entry pageflow) allow data entry by the end user.Â 
  • View forms (called view pageflow) and
  • Overview forms show data already entered; they can not be modified (available only in User XP Case History or via a Process Web app URL). They are generally used by the Process administrator, but they are added to your process in the same way as entry forms.

At the task level, you can add Entry and View forms only.

Generally, it is a good design practice to have your process and step variables already defined before beginning form design.
