Create a new lane

(Category: Process Modeling)

Lanes are used to organize a process functionally and visually.

Lanes exist only within a pool. Transition flow can cross lane boundaries within a pool.

You can create multiple lanes in a pool by using drag and drop from the Bonita Studio palette. To create a lane in a pool, go to the Design palette to the left of the whiteboard. Click and drag the lane icon onto the pool.

A lane can be specifically associated with an actor or a group of actors, so that the same actor(s) perform all tasks within that lane. This assignment can be overridden on a task-by task basis.
