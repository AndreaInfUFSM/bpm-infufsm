Define global and local variables

(Category: Data management)

Variables can be defined :

  • globally (for the entire process): To define a global variable, click on the pool or lane and go to the details panel. > data pane.
  • locally (accessible only to a single task and its transitions): To define a local variable, click on a task and go to the details panel > data pane.

The data in these variables are persisted, that is, saved in the Bonita database.

Data can also be defined as transient at the task level only, primarily for use in manipulating data in forms. This data is not persisted.

You can also define some variable to be a single- or multiple-type.
