Special fields

(Category: Forms Modeling)

You can apply a number of characteristics to fields in a form, using widgets in Bonita Studio form builder, including:

  • Tools tips and other user aids
  • Mandatory and read-only fields
  • When fields should appear in a form
  • Validators on field contents
  • Multi-instantiation of fields
  • Using complex data in fields
  • Using connectors to fill in initial data and to update data
