Check BPMN model

(Category: Validation)

As you design a process, when a task in a process model is incomplete (for example, missing an actor, a connector configuration, etc), a red X appears in the upper right corner of the element. This indicates that there is a correction needed.

You can deactivate this feature so you won't see the X marks in your process elements, which is useful when you want to share a clean diagram. Reactivate it when you want to see if there are missing configurations in the process.
