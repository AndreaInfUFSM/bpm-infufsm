Basic actions

(Category: Process Modeling)

A number of design tools exist in Bonita Studio which are used to modify and optimize the layout of the interface and the elements on the whiteboard. Design maniplation tools include:

  • the marquee tool for mulitple selection
  • the zoom in and out tool for close up viewing
  • the alignment tool for horizontal and vertical aligning
  • the spacing tool for horizontal and vertical spacing
  • the active plus signs for enlarging and reducing pool size
  • the arrows for displaying design the design palette
  • the arrows for modifying the size of menus
  • the standard copy/paste shortcuts for moving elements on the whiteboard
  • the standard undo/redo shortcuts for unlimited undoing/redoing of design actions saved temporarily in the history
