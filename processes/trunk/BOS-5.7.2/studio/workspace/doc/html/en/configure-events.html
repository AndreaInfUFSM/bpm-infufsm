Configure events

(Category: Process Modeling)

Events in BPMN2 include:

  • start events, which begin a process
  • end events, which terminate a process
  • intermediate events, which are embedded in a process flow
  • boundary events, which are attached to a task or event subprocess

Among these types of events are

  • messages (start, end, intermediate, boundary)
  • timers (start, intermediate boundary)
  • signals (start, intermediate, boundary)
  • errors (start, boundary, end)
  • links (intermediate)
