Define an organization

(Category: Organization)

An organization consists of:

  • Groups: a subunit of an organization (for example: R&D, Services, Marketing)
  • Roles: a function in a group (for example: developer, tester, manager, consultant, sales rep)
  • Users: an individual (for example, J.D. Mugwump)

You can define multiple organizations in Bonita Studio, and select one to apply to the process when you configure it just before run or export.

A process can be associated with only a single organization when it is run or exported.

An organization can be replaced or edited in Bonita User Experience.
