Test a connector

(Category: Connectors)

You can test the function of some types of connectors while still in design/development mode.

Check external configurations before testing a specific connector. Sometimes the connector test returns an error due to external factors, for example if a database connection is not open.

Not all connectors can be tested before executing the process. For example, some Bonita Connectors may return errors because the Bonita Execution Engine has not been launched (ie the process has not been Run).

Where dynamic values have been used, you will need to replace the Groovy expression with a static value to complete the test as there will be no data to fill the dynamic values.
