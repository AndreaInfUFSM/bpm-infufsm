Process simulation

(Category: Process Simulation)

Use simulation to evaluate the way a process runs under different resource availabilities and different load profiles.

When a simulation is run, a specified number of iterations over a specified period of time are run either with simulated data or with assigned probabilities, and the cumulated result of all iterations is shown in a report.

To use the simulation function on a process, first define simulation parameters on individual elements (tasks), then define simulation resources and load profiles.

There is an example process provided with BOS called Delivery-Simulation, which illustrates how to configure a simulation, using two processes with the same tasks and simulation parameters but slightly different flows. You can run both to see and compare the simulation reports.
